# Logo Assets

This directory should contain the following logo files:

## Required Files

1. **logo-horizontal-dark.png** (or .svg)
   - For: Header (light backgrounds), Admin panel
   - Variant: Deep purple or black on white
   - Recommended size: 180x40px minimum

2. **logo-horizontal-light.png** (or .svg)
   - For: Footer (dark backgrounds)
   - Variant: White or soft beige on dark
   - Recommended size: 180x40px minimum

3. **logo-vertical-dark.png** (or .svg) (optional)
   - For: Vertical contexts if needed
   - Variant: Deep purple or black on white
   - Recommended size: 120x200px minimum

4. **logomark.png** (or .svg)
   - For: Admin panel header, favicon source
   - Variant: Icon only, deep purple on white
   - Recommended size: 32x32px minimum

5. **favicon.ico**
   - For: Browser tab icon
   - Variant: Logomark or simplified icon
   - Size: 16x16, 32x32, 48x48 (multi-size ICO)

6. **apple-touch-icon.png** (optional)
   - For: iOS home screen icon
   - Size: 180x180px

## Source Files

The original logo files are located in the assets folder. Copy the appropriate variants:

- `FULL_LOGO_HORIZONTAL_DEEP_PURPLE_ON_WHITE` → `logo-horizontal-dark.png`
- `FULL_LOGO_HORIZONTAL_WHITE_ON_BLACK` → `logo-horizontal-light.png`
- `LOGOMARK_DEEP_PURPLE_ON_WHITE` → `logomark.png`

## Notes

- Prefer SVG format for scalability
- Ensure all logos maintain aspect ratio
- Test logos on both light and dark backgrounds
- Verify accessibility (contrast ratios)
